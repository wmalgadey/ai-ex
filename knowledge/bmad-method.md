## Was ist es?

**BMad = Build More Architect Dreams** — Ein AI-getriebenes agiles Entwicklungs-Framework (v6).

> "Breakthrough Method for Agile AI Driven Development"

Kein typisches AI-Tool das für dich denkt — BMad-Agenten agieren als **Expert Collaborators**, die durch strukturierte Prozesse führen und das eigene Denken in Partnerschaft mit der KI herausholen.

100% free & open source. Keine Paywalls.

## Kernfeatures

- **Scale-Domain-Adaptive** — passt Planungstiefe automatisch an Projektkomplexität an
- **Structured Workflows** — 34+ Workflows, grounded in Agile Best Practices
- **Specialized Agents** — 12+ Domain-Experten: PM, Architect, Developer, UX, Scrum Master, ...
- **Party Mode** — Mehrere Agent-Personas in einer Session für Diskussionen
- **Complete Lifecycle** — Von Brainstorming bis Deployment
- **/bmad-help** — AI-Guidance für "was als nächstes?"

## Quickstart

```bash
npx bmad-method install
# Dann AI-IDE (Claude Code, Cursor, etc.) im Projektordner öffnen
```
